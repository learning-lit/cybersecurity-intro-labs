Subnetting Cheat Sheet

A /22 subnet mask uses 22 network bits and leaves 10 host bits. Mask representation: 255.255.252.0.

• Usable Hosts: 2(32 - CIDR) - 2

• Network Address: Perform a bitwise AND between the binary representation of the IP and the mask. (Note: block size in the 3rd octet is 4).

• Broadcast Address: Set all host bits in the address range to binary 1.

TCP Header Field Decoder
Protocol Reference Cards

1. Hexadecimal numbers are converted to base-10 decimals. Example: 0x0050 = 80.

2. Byte 12 contains the Data Offset, while Byte 13 (represented by `12`) holds the raw TCP Flag bits.

3. Binary breakdown of Byte 13 flags: 0x12 in hex equals binary 00010010. This corresponds to the SYN and ACK flags being set.
